# Plan FIRST — Run Less, Run Faster

Generador de planes de entrenamiento basado en el método FIRST (Furman Institute)
del libro *Run Less, Run Faster*, portado desde la hoja de cálculo FIRST 2.2 de
foroatletismo.com.

Introduce tu marca (5K, 10K, media o maratón), elige la distancia a entrenar, la
fecha de inicio y la distribución semanal, y obtén el plan completo semana a
semana con tus ritmos: series, carrera corta (tempo) y carrera larga, más
predicción de tiempos de carrera. Permite registrar los resultados de cada
entrenamiento (tiempos, FC, sensaciones); se guardan en el navegador y pueden
exportarse/importarse como JSON.

App de un solo archivo, sin dependencias: [index.html](index.html).
